# DepotKontroll
Python and MySQL based Sensor script for climate variables.

# Equipment
- Tsys01 Temperature sensor
- BME680 Air quality sensor
- Waveshare 2.13inch E-ink screen
- Waveshare ETH/USB HUB HAT
- Raspberry Pi Z

# Requirements to get started
Pip
i2c configuration
MySQL

# Directory for clone
/home/pi/bin/$(repo)
